Cyrus UI is a React component library designed with Tailwind CSS, providing a collection of customizable and reusable UI components. This library aims to simplify the development process by offering a set of pre-styled components that you can easily integrate into your React applications.

## Features

-   **Tailwind CSS Integration**: Built with Tailwind CSS for easy customization and responsive design.
-   **Reusable Components**: A variety of components that can be reused across your projects.
-   **Customizable Styles**: Every component can be easily customized using Tailwind CSS classes, allowing you to tailor the look and feel to match your application's design.
-   **Easy to Use**: Designed with simplicity in mind, making it easy to integrate into your existing React applications.

## Installation

To install the Cyrus UI package, you can use npm:

```bash
npm install cyrus-ui
```

## Usage

Here's a simple example of how to use a component from the Cyrus UI library:

```typescript
import { Button } from "cyrus-ui";

const App = () => {
    return (
        <div>
            <Button variant="primary" size="small">
                Click me!
            </Button>
        </div>
    );
};
export default App;
```

## Customization

You can customize every component using Tailwind CSS classes. For example, you can change the background color, text color, padding, margins, and more by simply adding or modifying Tailwind utility classes.
